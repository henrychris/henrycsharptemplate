# HenryCsharpTemplate

## Description

This is a template for projects in C#. It is meant for a single web API project.

## Installation

Clone the repository  

```bash
git clone https://github.com/henrychris/henrycsharptemplate.git
```

cd to the root directory  

```bash
cd henrycsharptemplate
```

Install the template

```bash
dotnet new install .
```

## Usage

Create a new project

```bash
dotnet new henrycsharptemplate -n YourProjectName
```

You can run the project using the `dotnet run` command in the HenryCsharpTemplate folder. The project will be available at `http://localhost`.

Access `http://localhost/swagger` to see the available endpoints.
